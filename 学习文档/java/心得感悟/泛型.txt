泛型
字面意思，广泛的类型。专业解释，参数化类型，即类型不再固定，将类型作为参数进行传导。

作用：
1.将类型参数化，同一个代码可以实现多种类型的使用，代码复用。
2.泛型中的类型在使用时指定，不需要强制类型转换（类型安全，在编译器阶段就可以检查是否报错）


泛型三种使用方式：
1.泛型类   public class GenericClass<T> {}
2.泛型接口 public interface Generator<T> {}
3.泛型方法  private static<T> T genericAdd(T a, T b) {}
